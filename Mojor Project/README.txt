                                   **** HOW TO INSTALL EMART ON YOUR PC ****

1) Please copy this folder(emart) to your htdocs or if you are using wamp then copy it to www directory.
2) Please import this sql file in phpmyadmin (best work with xampp)
3) How to import sql file?
   -> Open phpmyadmin (using localhost)
   -> Create a new database named 'emart'
   -> click on database name from left menu
   -> goto import option right in the middle of the screen.
   -> browse the file .
   -> and just click on go .
   -> wait for few minutes and the databse will be ready.
4) Just go and access the emart using link localhost/emart OR localhost/emart/index.php
